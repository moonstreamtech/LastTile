# Monetization Plan (Fair-Play First)

## Shipped in v0.1
- Single fixed-height bottom banner ad slot (50 dp, full-width) pinned below the gameplay area. Currently a placeholder composable; swap for a real `BannerAd` view when the ad SDK lands.
- No interstitials, no rewarded ads, no in-app purchases.

## Rewarded ad opportunities (planned)
1. **One-time revive per run** after game over.
2. **Post-run reward doubler** (optional XP bonus).
3. **Daily challenge reroll** (limited usage).

## Cosmetic monetization ideas (planned)
- Board themes (neon, midnight, pastel contrast)
- Tile skins and particle trails
- UI accent packs

## Principles to protect retention
- No pay-to-win stat boosts.
- No interstitials between runs; the only ad surface is the bottom banner.
- Ads are fully optional to the player's progression; core game remains playable offline with no account.
- Reward values should be helpful but not mandatory.
